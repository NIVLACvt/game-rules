Setup Instructions

1.To join on a java edition device:

- Open Minecraft and go to the "Multiplayer" screen.

- Click "Add Server" and enter what ever name you whant the server name to be called.

- Enter the server IP address.

- Click "Done" and then "Join Server".

2.To join on a nintendo switch:

- First go to the home screen then into "system settings".

- Then go to "Internet" tab and click "Internet settings".

- Then from there click on the network you are currently connected to and click "Change Settings".

- Then go down to "DNS Settings" and set it to "Manual".

- After you do that then you need to change the "Primary DNS" to 104.238.130.180 and the "Secondary DNS" to 8.8.8.8 then click Save.

- Once that is finished go back to the home screen of the switch.

- Now you can open minecraft(make sure you are logged into your microsoft account).

- Then click play and go to the "Servers" tab on the top of the screen.

- Then join the server called "The Hive".

- After clicking through the message of the day(there will not allways be one).

- You need to select "Connect to a Server".

- Then where is says to "Please enter the IP or address" enter [enter server IP here]

- Leave the port( or the next box) as is.

- In the last box with "(Optional) Display Name on Server List" obove it enter the name you want the server to be called

- Then toggle the "Add to server list" to ON, if you want ot have it saved to the server list.

- If you do not toggle the toggle switch to ON you will have to do this every time you want to join the server

- Then click "Submit" and you will join the server!!!

warning : To join on nintendo switch you will need a microsoft account and nintendo online
